# DTunnel SDK Demos

Demos prontas geradas pelo CLI oficial (`dtunnel-sdk init`):

- `demos/cdn`
- `demos/typescript`
- `demos/react-typescript`

Cada demo ja vem com `build:android` para gerar um arquivo final unico:
- `dist/build.html`

## Como usar

CDN:

```bash
cd demos/cdn
npm install
npm run build:android
```

TypeScript:

```bash
cd demos/typescript
npm install
npm run dev
npm run build:android
```

React + TypeScript:

```bash
cd demos/react-typescript
npm install
npm run dev
npm run build:android
```
